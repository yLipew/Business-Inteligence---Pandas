Este repositório contém um guia prático e técnico sobre a estrutura de dados **Series** da biblioteca Pandas, essencial para fluxos de trabalho em Ciência de Dados e Inteligência Artificial.

## 📌 Visão Geral

O arquivo `exemplos.ipynb` serve como um material de referência para a criação, indexação e manipulação de Series. O conteúdo aborda desde a inicialização básica até o tratamento de exceções e acessos avançados por rótulos.

## 🚀 Tópicos Abordados no Código

* **Criação de Series:** Inicialização de objetos vazios, a partir de listas, arrays NumPy e dicionários.
* **Indexação e Acesso:** Demonstração de acesso por posição (`iloc`) e por rótulo (`loc`).
* **Slicing:** Recuperação de múltiplos elementos (primeiros e últimos da sequência).
* **Tratamento de Dados:** Como o Pandas lida com elementos ausentes (`NaN`) e exceções de chave (`KeyError`).

## 🛠️ Tecnologias e Ferramentas

* **Linguagens e Web:** Python, Numpy, Pandas e Jupiter Notebook.
* **Data Intelligence:** Power BI para visualização de dados.
* **Inteligência Artificial:** Machine Learning e bibliotecas de manipulação de dados.
